# static/test/

Frontend tests for property-based validation and integration testing with Playwright.

## Files

| File | What | When to read |
| ---- | ---- | ------------ |
| `package.json` | NPM dependencies and test scripts (testdouble, fast-check, Playwright) | Understanding dependencies, running tests, adding test libraries |
| `app.test.js` | Property-based tests using testdouble and fast-check | Understanding test coverage, debugging state management |
| `integration.test.js` | Playwright integration tests for full browser automation | Validating end-to-end UI flows, browser compatibility |
| `index-html-manual-test.js` | Manual test helpers for HTML structure validation | Quick manual testing, debugging layout issues |

## Subdirectories

| Directory | What | When to read |
| --------- | ---- | ------------ |
| `fixtures/` | Test data and mock configurations | Understanding test data, adding new test cases |
