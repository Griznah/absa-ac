# static/js/

Frontend JavaScript bundles and vendor files.

## Files

| File | What | When to read |
| ---- | ---- | ------------ |
| `alpine.min.js` | Alpine.js v3.14.0 bundled locally (vendor file) | Verifying version, CSP compliance, debugging framework issues |
| `alpine.min.js.sha256` | SHA256 checksum for Alpine.js SRI validation | Verifying integrity, updating Alpine.js version |
| `app.js` | Alpine.js application logic with API client, reactive state, polling | Understanding state management, API integration, adding new features |
